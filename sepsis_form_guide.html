🚀 วิธีสร้างฟอร์มด้วย Script อัตโนมัติ (แนะนำ)
1

เปิด Google Apps Script

เปิดเบราว์เซอร์ แล้วไปที่ลิ้งค์นี้ (ต้อง login บัญชี Google ก่อน)

https://script.google.com
2

สร้าง Project ใหม่

คลิก "New project" (ปุ่มสีน้ำเงิน มุมซ้ายบน) จะเปิดหน้า editor โค้ด

3

วางโค้ด Script

ลบโค้ดเดิมทั้งหมดออก แล้ววางโค้ดจากไฟล์ sepsis_form_creator.gs ที่ดาวน์โหลดไว้

💡 Tip: กด Ctrl+A เพื่อเลือกโค้ดเดิมทั้งหมด แล้ว Ctrl+V วางโค้ดใหม่
4

Run Script

ตรวจสอบว่าเลือก function "createSepsisForm" แล้วกดปุ่ม ▶ Run

⚠️ Google จะขอ Permission ครั้งแรก — ให้คลิก "Allow" เพื่ออนุญาต
5

รับลิ้งค์ฟอร์ม

รอประมาณ 10–20 วินาที จะมี popup แสดงลิ้งค์ 2 อัน:

สิ่งที่ได้รับ:

  • ลิ้งค์ Google Form สำหรับ RN กรอกข้อมูล
  • Google Sheet บันทึกคำตอบอัตโนมัติ
  • Sheet "📊 Dashboard" พร้อมสูตรสถิติ
📤 วิธีแชร์ให้ RN
📱

แชร์ผ่าน LINE / QR Code

คัดลอกลิ้งค์ฟอร์ม → วางใน LINE กลุ่ม Ward
หรือสร้าง QR Code จากลิ้งค์ แปะที่ Nurse Station ได้เลย

💡 ใช้เว็บ qr-code-generator.com แปลงลิ้งค์เป็น QR Code ฟรี แล้วปริ้นติดที่ Ward
💻

ดูสถิติใน Google Sheets

ดูสถิติสรุปที่ Sheet "📊 Dashboard" ข้อมูลอัปเดตทันทีทุกครั้งที่ RN กด Submit

❓ คำถามที่พบบ่อย
Q: RN ต้องมีบัญชี Google ไหม?
A: ไม่จำเป็น ถ้าตั้ง Form ให้ "ไม่ต้อง login" (ค่า default) RN เปิดลิ้งค์กรอกได้เลย
Q: ข้อมูลจะหายไหม?
A: ไม่หายครับ ข้อมูลเก็บใน Google Drive ของ account ที่รัน script ข้อมูลสะสมตลอด
Q: ถ้าอยากให้คนอื่นดู Sheet ด้วย?
A: เปิด Google Sheet → Share → ใส่อีเมลหรือตั้งเป็น "Anyone with link can view"
Q: อยากเพิ่ม field ได้ไหม?
A: ได้เลย เข้าไปแก้ใน Google Forms ปกติ ข้อมูลใหม่จะเพิ่มเป็น column ใน Sheet อัตโนมัติ

🎉 พร้อมใช้งาน!
